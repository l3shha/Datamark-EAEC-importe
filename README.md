# Веб-приложение для ввода в оборот товаров из ЕАЭС

Веб-приложение для автоматизации процесса ввода в оборот обуви, импортированной из Российской Федерации, в белорусской системе маркировки «Электронный знак».

## Функциональность

1. Загрузка двух файлов из РФ:
   - Файл 1: GTIN + описание товара + количество
   - Файл 2: Неполные коды маркировки (без криптохвоста)

2. Автоматизация процесса:
   - Парсинг загруженных файлов
   - Авторизация в API ГИС «Электронный знак»
   - Заказ недостающих частей кодов маркировки (криптохвосты)
   - Ожидание выполнения заказа с проверкой статуса
   - Скачивание полных кодов
   - Отправка отчета о вводе товаров в оборот
   - Отслеживание статуса отчета

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` в корне проекта:
```
API_BASE_URL=https://sandbox-api.datamark.by
API_USERNAME=your_username
API_PASSWORD=your_password
```

3. Запустите приложение:
```bash
python app.py
```

4. Откройте браузер и перейдите на `http://localhost:5000`

## Структура проекта

- `app.py` - основной файл Flask приложения
- `file_parser.py` - модуль для парсинга файлов
- `api_client.py` - модуль для работы с API ГИС «Электронный знак»
- `templates/` - HTML шаблоны
- `static/` - CSS и JavaScript файлы
- `.env` - конфигурация (не включен в репозиторий)
