# Преобразование кодов маркировки РФ в РБ стандарт

Приложение для преобразования российских кодов маркировки в белорусский стандарт путем замены кода страны в серийном номере.

## Функциональность

- Преобразование кодов: замена первой цифры после '21' с 5 (РФ) на 2 (РБ)
- FastAPI endpoint для обработки списка кодов
- Скачивание результата в виде текстового файла

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python main.py
```

3. API будет доступен на `http://localhost:8000`

## API Endpoints

### POST /convert
Преобразует список кодов и возвращает JSON с результатом.

**Запрос:**
```json
{
  "codes": [
    "0104811644010685215mpid7gn7wzu...",
    "0104660575291478215GQNjOY>S3!sQ"
  ]
}
```

**Ответ:**
```json
{
  "converted_codes": [
    "0104811644010685212mpid7gn7wzu...",
    "0104660575291478212GQNjOY>S3!sQ"
  ]
}
```

### POST /convert/download
Преобразует список кодов и возвращает файл для скачивания.

**Запрос:** Тот же JSON.

**Ответ:** Файл `converted_codes.txt` с преобразованными кодами (по одному на строку).
